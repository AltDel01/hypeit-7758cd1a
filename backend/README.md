# ai-content-backend
Backend service for AI-powered content planning, PDF export, and media generation.

## Tech Stack
- Node.js
- NestJS
- Google Gemini API
- BytePlus / Seedream API
- Swagger (OpenAPI)

## Setup
1. Clone repository
2. Copy .env.example to .env
3. Fill API keys
4. npm install
5. npm run start:dev
